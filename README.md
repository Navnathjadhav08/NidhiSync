## Installation ğŸ’»

### Prerequisites
- Node.js 16+
- MongoDB 4.4+
- Redis 6.2+

### Setup Steps

1. Clone the repository:
```bash
git clone https://github.com/Navnathjadhav08/NidhiSync.git
cd NidhiSync
```

2. Install dependencies:
```bash
npm install
```

3. Create environment file:
```bash
cp .env.example .env
```

4. Update `.env` with your configuration values

5. Start the development server:
```bash
npm run dev
```
